## Task 2-4：更新合成逻辑

**修改文件**：`client/assets/scripts/core/MergeManager.ts`

**内容**：
- 两个相同等级甜品碰撞 → 销毁两个，在碰撞点生成下一等级
- 两个 Lv8 碰撞 → 销毁两个，+50 金币，+1000 分
- 合成时触发 Juice 反馈：
  - 普通合成：缩放弹跳 + 小粒子爆发 + 叮咚音效
  - 连锁合成：连击数字 + 屏幕轻震 + 音效 pitch 递增
  - Lv8 消除：全屏闪白 + 大粒子 + 金币飞散 + 华丽音效
- 合成得分按指数递增表计算

**验收标准**：
- [ ] 同级碰撞正确合成为下一级
- [ ] Lv8 碰撞正确消除 + 奖励
- [ ] 合成粒子特效和音效正常播放
- [ ] 连锁合成有递增反馈

## Task 2-5：实现合成即飞送餐机制

**修改文件**：
- `client/assets/scripts/core/MergeManager.ts`
- `client/assets/scripts/core/CustomerManager.ts`

**内容**：

**验收标准**：
