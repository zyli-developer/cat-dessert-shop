## 游戏概述

一款竖屏合成休闲小游戏。玩家在甜品店的玻璃容器中投放甜品元素，相同元素碰撞合成为更高级甜品（类似合成大西瓜）。左侧有猫咪顾客点单，合成出目标甜品后自动送给顾客。服务完当前回合所有顾客即通关。

**核心卖点**：合成物理玩法 + 餐厅经营目标 + 可爱猫咪角色

---

## 目标用户画像

**主要用户：猫咪爱好者**
- 男女不限，16-30 岁，喜欢猫咪相关内容
- 在抖音关注猫咪博主、看猫咪视频，会因为"猫咪"主题点进来
- 玩休闲小游戏，不玩重度游戏
- 核心动力：收集猫咪品种、装饰猫咪家园、给好友"炫耀"自己的猫
- 愿意为解锁新品种猫咪持续通关

**次要用户：休闲玩家**
- 被合成玩法本身吸引，从分享链接进入
- 留存靠玩法，猫咪主题是加分项

## 三层核心循环

| 循环层 | 内容 |
|--------|------|
| **30 秒（微观）** | 滑动选位 → 松手掉落 → 物理碰撞 → 合成反馈。随机掉落 + 位置判断，低操控高随机的爽快手感 |
| **30 分钟（中观）** | 开局 → 投放合成 → 服务顾客 → 结算（星级 + 猫币 + 本关好友排名）→ 再来一局（冲排名/冲三星/好奇下一关需求）。单局时长自然增长：前期 1-2 分钟，后期 3-8 分钟 |
| **30 小时（宏观）** | MVP：三星收集 + 好友分数超越 + "家园即将开放"预告。v2.0+：家园养猫 + 猫咪品种收集 + 好友串门 |
