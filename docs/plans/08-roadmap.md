# 猫咪收集体系与版本路线

> 所属项目：猫咪甜品店
> 总文档：[README.md](README.md)

---

## 1. 猫咪品种解锁（v2.0 规划）

通关解锁资格 + 猫币购买：

| 解锁条件 | 猫咪品种 | 猫币价格（暂定） |
|---------|----------|----------------|
| 默认 | 橘猫 | 免费 |
| 通关第 3 关 | 英短蓝猫 | 50 |
| 通关第 7 关 | 布偶猫 | 100 |
| 通关第 10 关 | 暹罗猫 | 150 |
| v1.2+ 更多关卡 | 更多品种... | 递增 |

---

## 2. 家园系统（v2.0 规划）

- 每只猫咪可在家园中展示
- 猫币购买家具装饰家园
- 好友可以串门查看彼此的家园和猫咪
- 家园入口在 MVP 主页预告（灰色"家园"按钮 + "即将开放"）

---

## 3. 宏观循环闭环

> 通关解锁猫咪品种 → 猫币购买 → 家园养猫 → 好友串门看猫 → 想要更多品种 → 继续通关

---

## 4. 后续版本路线

### v1.1 — 变现增强
- 关卡间插屏广告（每 3 关一次）

### v1.2 — 内容扩充
- 新手手指引导（3-4 步）
- 关卡扩充至 20 关
- 补充完整音效（8 种）
- 新增猫咪品种（随关卡扩充）

### v1.3 — 社交与美术
- 猫咪随机组合外观（颜色/衣服/配饰）
- 录屏分享（`tt.getGameRecorderManager`）
- 好友排行优化

### v2.0 — 家园与收集
- 家园系统上线（猫咪展示 + 家具装饰）
- 猫咪品种收集（通关解锁 + 猫币购买）
- 好友串门
- 猫咪皮肤

### v2.1 — 深度扩展
- 顾客耐心值（高回合启用）
- 每日签到 / 成就系统
- 更多道具种类
- 限时活动关卡
